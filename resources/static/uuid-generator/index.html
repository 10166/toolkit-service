UUID版本参考

版本 名称 生成方式 特点 应用场景
UUID v1 基于时间 时间戳 + MAC地址 包含时间信息，有序性 需要时间顺序的场景
UUID v3 基于MD5 命名空间 + 名称 + MD5 确定性生成 从名称生成唯一ID
UUID v4 随机 随机数或伪随机数 完全随机，无序 通用唯一标识符
UUID v5 基于SHA-1 命名空间 + 名称 + SHA-1 确定性生成，更安全 需要安全性的场景
Nil UUID 全零UUID 所有位都为零 特殊UUID，表示空值 表示空或未设置
